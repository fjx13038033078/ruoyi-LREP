# 学习资源分享平台用户端

## 项目介绍
这是一个基于Vue.js的学习资源分享平台用户端，使用了Element UI组件库，提供了丰富的学习资源浏览、搜索、下载等功能。

## 功能特性
- 用户登录注册
- 资源浏览和搜索
- 资源分类查看
- 用户个人中心
- 资源收藏
- 资源评论和评分
- 提供美观的界面设计

## 开发环境
- Node.js >= 10.0.0
- npm >= 6.0.0

## 运行项目
```bash
# 安装依赖
npm install

# 开发环境运行
npm run dev

# 生产环境打包
npm run build:prod
```

## 项目结构
```
src/
├── api/            # API接口
├── assets/         # 静态资源
├── components/     # 组件
├── router/         # 路由
├── store/          # Vuex存储
├── utils/          # 工具函数
├── views/          # 页面
├── App.vue         # 根组件
└── main.js         # 入口文件
```
